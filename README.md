### Описание проекта
Мы представляем наш телеграм-бот для организация досуга. В данном боте Вы можете подобрать досуг конкретно под ваше место/бюджет/время.
Наш бот поможет Вам быстро найти что-то интересное, так как собирает огромное количество данных с различных источников и выдает Вам готовую выжимку из мероприятий.
Остается только выбрать!

## ⚡ Быстрый старт через терминал

### 1. Клонируйте репозиторий и перейдите в папку проекта

```bash
cd путь/до/папки/проекта
```

### 2. Создайте и активируйте виртуальное окружение

```bash
# Создание
python -m venv venv

# Активация на Windows
venv\Scripts\activate

# Активация на macOS / Linux
source venv/bin/activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Укажите переменные окружения

```bash
# Windows (Command Prompt)
set DB_HOST=ваш_хост
set DB_PORT=5432
set DB_NAME=имя_базы
set DB_USER=пользователь
set DB_PASSWORD=пароль

# macOS / Linux
export DB_HOST=ваш_хост
export DB_PORT=5432
export DB_NAME=имя_базы
export DB_USER=пользователь
export DB_PASSWORD=пароль
```



### 5. Запустите проект

```bash
python run_all.py
```

## 🛠 Запуск через PyCharm

1. Заполните данные хоста в файле `.env`
2. Установите соединение между хостом и PyCharm *(см. раздел ниже)*
3. Установите библиотеки из `requirements.txt` через терминал PyCharm (`Alt+F12`):

```bash
pip install -r requirements.txt
```

4. Запустите файл `run_all.py`

---

## 🔗 Установка соединения между хостом PgAdmin4 и PyCharm

1. В верхнем левом углу откройте **Main Menu** (значок ☰)
2. Выберите **Run → Edit Configurations**
3. В появившемся окне выберите файл `run_all.py`
4. В поле **Environment variables** нажмите на значок 📄 (в правом конце строки)
5. Через кнопку **+** введите данные вашего хоста, нажмите **OK**

---

## 🔍 Где найти данные хоста в PgAdmin4?

1. Откройте **PgAdmin4**
2. Выберите нужный хост → правая кнопка мыши → **Properties**
3. Перейдите на вкладку **Connection**

> Здесь хранятся все данные вашего хоста.

## 📦 Установка PgAdmin4

PgAdmin4 — инструмент управления базой данных. Для его работы сначала нужно установить **PostgreSQL**.

### Установка PostgreSQL

1. Перейдите на [postgresql.org](https://www.postgresql.org/)
2. Выберите **Download → вашу ОС → Download the Installer**
3. В таблице выберите нужную версию и ОС, запустите дистрибутив
4. В инсталляторе:
   - Выберите путь установки
   - Выберите компоненты *(рекомендуется оставить по умолчанию)*
   - Выберите директорию для данных
   - Нажимайте **Next** до завершения установки
5. После установки **снимите галочку** с пункта о запуске StackBuilder, нажмите **Finish**

### Первый запуск PgAdmin4

1. Откройте **PgAdmin4**
2. Во всплывшем окне придумайте и введите **мастер-пароль**

🎉 Поздравляем — ваш хост создан!
